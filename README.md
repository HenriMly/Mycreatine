# Mycreatine

Mycreatine est une application web (Angular) de type **boutique en ligne de compléments alimentaires**.

L'objectif du projet est de servir de base pour un shop (catalogue produits, pages, évolutions futures), avec une stack moderne et un setup simple à lancer.

## Stack

- Angular (Angular CLI)
- TypeScript
- Tests unitaires via `ng test`

## Prérequis

- Node.js + npm

## Installation

```bash
npm install
```

## Démarrer en local

```bash
npm start
```

Puis ouvrir http://localhost:4200/

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```
